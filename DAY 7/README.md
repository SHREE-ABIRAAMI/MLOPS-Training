# DIGITAL MIRROR

AI-Powered Personality & Behavioral Analysis System

## Overview
Digital Mirror is a premium Streamlit SaaS application built for behavioral risk analysis using a Random Forest classifier. The app is designed with a futuristic glassmorphism interface and advanced analytics features.

## Features
- Landing page with premium UI
- Interactive dataset explorer with AG Grid support
- High-fidelity Plotly visualizations
- Prediction center with dynamic input generation
- Model performance and feature importance dashboards
- AI-driven insights and behavioral interpretation

## Installation
1. Create a Python environment
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Dataset
The app uses `digital_mirror_research_dataset_870_rows.csv` located in the project root.
